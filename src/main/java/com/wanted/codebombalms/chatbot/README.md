# Chatbot Domain

`chatbot` 도메인은 학습자가 문제 풀이 중 AI 챗봇과 대화하도록 채팅방·메시지·AI 응답 생성을 관리한다.
내부 채팅 기록과 외부 FastAPI AI 서버 연동을 함께 담당한다.

## 주요 역할

- 첫 메시지 전송과 동시에 새 채팅방을 생성한다.
- 채팅방에 사용자 메시지를 추가하고 AI 응답을 저장한다.
- 채팅방 목록·메시지 이력을 조회하고, 채팅방을 삭제한다.
- 문제 세트·문제·진행률·이전 대화를 조합해 AI 요청 컨텍스트를 만든다.

## 주요 모델 (용어)

| 모델 | 설명 |
| --- | --- |
| `ChatRoom` | 사용자와 문제 세트 기준으로 생성되는 대화방 |
| `ChatMessage` | 채팅방에 저장되는 사용자/AI 메시지 |
| `MessageRole` | 메시지 작성 주체(사용자/AI)를 구분하는 역할 값 |
